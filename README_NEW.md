## 🚀 Quick Start

### 1. Install (5 minutes)
```bash
# Clone project
git clone <repo-url>
cd autonomous-ai-system

# Setup
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Install
pip install -r requirements.txt
```

### 2. Run
```bash
python -m src.main
```

### 3. Choose Mode
- **Autonomous** (Recommended): Let it learn automatically
- **Interactive**: You control operations

## 🔑 Key Features

### 🧠 Autonomous Learning
- Continuously crawls the web for knowledge
- Learns from diverse sources automatically
- Extracts concepts and patterns
- No human intervention needed

### 🚀 Self-Improvement
- Analyzes its own performance
- Identifies improvement opportunities
- Implements code changes
- Measures impact and adopts winners

### 💭 Intelligent Reasoning
- Plans multi-step solutions
- Evaluates options with confidence scoring
- Makes data-driven decisions
- Handles uncertainty gracefully

### 💾 Persistent Memory
- Short-term working memory (TTL)
- Long-term episodic memory (persistent)
- Skill and experience tracking
- Automatic memory consolidation

### ♾️ Indefinite Operation
- Runs 24/7 without human intervention
- Autonomous recovery from failures
- Health monitoring and auto-repair
- State persistence across restarts


## 🔧 Configuration

Edit `src/config.py` or use `.env`:

```python
# Autonomous mode
AUTONOMOUS_MODE_ENABLED = True
AUTONOMOUS_ITERATIONS = 10  # cycles per run

# Web crawler
MAX_CRAWL_DEPTH = 3
MAX_PAGES_CRAWL = 100

# Learning
BATCH_SIZE = 32
LEARNING_RATE = 0.001

# Memory
SHORT_TERM_TTL = 3600  # 1 hour
MAX_MEMORY_GB = 4
```

See [Configuration Reference](docs/reference/configuration.md) for all options.
