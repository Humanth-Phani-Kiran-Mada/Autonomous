## How It Works

### Decision Flow

```
User Request
    ↓
DecisionContext (GAME, LOTTERY, STRATEGY, etc.)
    ↓
FastDecisionEngine Routes Request
    ↓
Selects Engine:
├─ EvolutionaryDecisionMaker (for STRATEGY, OPTIMIZATION)
├─ ProbabilisticPredictor (for LOTTERY, PREDICTION)
└─ GamePlayingEngine (for GAME)
    ↓
Engine Processes (milliseconds)
    ↓
Decision Result:
├─ choice: The selected option
├─ confidence: 0.0-1.0 confidence score
├─ reasoning: Explanation of choice
├─ alternatives: Other good options
└─ execution_time_ms: How long it took
```

### Genetic Algorithm Evolution

```
Initialize Population (50-200 chromosomes)
    ↓
Evaluate Fitness (for each chromosome)
    ↓
Tournament Selection (pick best traits)
    ↓
Crossover & Mutation (genetic recombination)
    ↓
Repeat Generations (5-50 generations in milliseconds)
    ↓
Return Best Solution (with confidence score)
```

## Usage Examples

### Example 1: Make a Fast Strategic Decision

```python
from src.evolutionary_decision_engine import get_fast_decision_engine, DecisionContext

engine = get_fast_decision_engine()

# Make a strategic decision
decision = engine.make_fast_decision(
    DecisionContext.STRATEGY,
    {
        "traits": ["speed", "accuracy", "cost"],
        "fitness_function": lambda x: (
            x["speed"] * 0.4 +
            x["accuracy"] * 0.4 +
            x["cost"] * 0.2  # Minimize cost
        ) * 100
    },
    max_latency_ms=20
)

print(f"Best choice: {decision.choice}")
print(f"Confidence: {decision.confidence:.1%}")
print(f"Latency: {decision.execution_time_ms:.3f}ms")
```

### Example 2: Predict Lottery Numbers

```python
from src.evolutionary_decision_engine import get_fast_decision_engine, DecisionContext

engine = get_fast_decision_engine()

# Predict next lottery number

prediction = engine.make_fast_decision(
    DecisionContext.LOTTERY,
    {
        "historical_sequence": historical,
        "possible_values": possible
    },
    max_latency_ms=5
)

print(f"Predicted: {prediction.choice}")
print(f"Confidence: {prediction.confidence:.1%}")
```
